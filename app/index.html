Scale Size

Design with math.

What is this?

Scale Size is a playground for calculating web-friendly modular scales. Punch in some numbers relevant to your project, and generate a harmonious type scale to use throughout your design.

Tips and tricks

  • Start with a base font size for body text. This is usually where paragraph text feels the most comfortable to read at length.
  • Choose an alternate size from a good-looking heading or title. Maybe even pick a second base font size to use on a responsive website.
  • Experiment! You never know what’s going to look good ahead of time, so have fun and try a bit of everything.
